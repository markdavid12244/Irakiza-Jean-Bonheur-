# Irakiza Bonheur- Personal Portfolio

This is my personal portfolio website built as part of the Advanced Web Design and Development assignment. It showcases my skills, projects, and provides a way to contact me.

## Features
- Semantic HTML5
- Responsive design with Flexbox and Grid
- Dark mode toggle
- Interactive calculator (Musanze Transport Fare Estimator)
- Downloadable CV (TXT)
- Contact form with validation
- Dynamic year in footer

## AI Usage Declaration
- **What I asked AI**: I used AI to debug CSS issues, get ideas for the calculator logic, and understand how to implement a dark mode toggle.
- **What I implemented myself**: I wrote all the HTML structure, styled the components, and wrote the JavaScript for the calculator, form validation, and download feature. I also designed the layout and content.
- **What I modified**: I adjusted the AI-suggested code to fit my design, fixed spacing, and ensured all requirements were met.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (DOM manipulation, events)
- Font Awesome for icons
